`git commit` した時に以下のエラーが出た。

```bash
error: gpg failed to sign the data
fatal: failed to write commit object
```

`git commit -S` しても変わらず。

以下も試したけど解消せず。

https://qiita.com/goahi/items/493ff01b8fc86539f885

```bash
export GPG_TTY=$(tty)
```

最終的にこれで解消した。

```bash
killall gpg-agent
echo "test" | gpg --clearsign
```

その後にもう一度 `git commit` したら通った。
